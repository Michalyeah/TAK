Oto przykładowy test, który zawiera różne elementy HTML. Teskt może być również zapisany
kursywą lub z naciskiem. Czasem warto podkreślić ważne informacje. Jeśli chcesz, możesz użyć
mniejszego tekstu, gdy coś jest mniej istotne. Kiedy coś zostaje usunięte, możesz użyć skreślenia, a nowy
teskt wstawić. Czasem potrzebne są też indeksy dolne, jak H2O, lub górne, jak E=mc2.
Można także dodać listę :
  1. Element pierwszy
  2. Element drugi
  3. Element trzeci
Lub listę wypunktowaną:
  • Punkt pierwszy
  • Punkt drugi
  • Punkt trzeci
